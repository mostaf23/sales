



 AdventureWorks Sales Analysis Dashboard

📌 Project Overview

This project analyzes **AdventureWorks sales data** using **Power BI** to build an interactive dashboard that helps understand sales performance, customer behavior, and product profitability.

The project includes **data modeling, data transformation, and business insights extraction** to support data-driven decision making.

The dashboard provides an overview of:

* Revenue performance
* Total orders
* Profit analysis
* Product performance
* Geographic distribution of sales
* Customer purchasing behavior

---

# 🧩 Data Modeling

The dataset was transformed into a **Star Schema model** to improve performance and simplify analysis.

### Fact Table

**Sales**

Contains transactional data such as:

* OrderID
* ProductKey
* CustomerKey
* OrderDate
* Revenue
* Profit
* Return quantity

### Dimension Tables

* Customers
* Products
* Product Category
* Product Subcategory
* Territories
* Calendar

Relationships were created between the **fact table and dimension tables** to support efficient reporting and filtering.

# ❓ Business Questions

This analysis aims to answer the following questions:

1. What is the **total revenue and profit generated by the business?**

2. Are we **meeting the monthly revenue targets?**

3. How do **weekday orders compare to weekend orders?**

4. What are the **best selling product categories and subcategories?**

5. Which **products generate the highest revenue?**

6. What is the **geographic distribution of sales and returns?**

7. How do **sales trends change over time?**

8. Which products have the **highest return rates?**

---

# 📈 Key Insights

### 💰 Revenue Performance

* The business generated **13.56K in total revenue**.
* Revenue growth fluctuates throughout the months.

### 📦 Orders Analysis

* Total orders reached **5430 orders**.
* Orders are **significantly higher during weekdays compared to weekends**.

### 🏆 Product Performance

* **Accessories** generated the highest number of orders.
* **Bikes** contribute significantly to revenue despite lower order volume.

### 🌍 Geographic Insights

* The **United States** has the highest number of product returns.
* Strong sales activity in **North America and Europe**.

 🔁 Returns Analysis

* Some products show **higher return rates**, which may indicate product issues or customer dissatisfaction.

📈 Trend Analysis

* Sales show **seasonal patterns**, with some peaks across different months.
